\doxysubsection{Detailed Description}
Implementation of the Node structure for DESERT. 

The \doxylink{classDesertNode}{Desert\+Node} class is used to keep track af all the entities created by a specific node. Each of them is stored in a vector of pointers to the original memory locations mainly used to provide discovery functionalities.
