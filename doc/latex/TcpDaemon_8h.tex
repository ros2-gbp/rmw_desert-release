\doxysubsection{Detailed Description}
Class used to send and receive data from the DESERT socket. 

The DESERT protocol stack interacts with the application level through a socket, used to send and receive a binary stream containing packets. This class connects to the socket and creates two threads, that run continously to store and send packets in the static members rx\+\_\+packets and tx\+\_\+packets
