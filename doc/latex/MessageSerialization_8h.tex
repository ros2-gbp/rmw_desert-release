\doxysubsection{Detailed Description}
Namespace containing serialization functions. 

The message data structure coming from upper layers is interpreted using type support informations passed by ROS2 during the creation of publishers, subscribers, clients and services. Those functions are used to compute the exact position that every data type must assume in memory an then calls Tx\+Stream or Rx\+Stream to receive or write them in the assigned location.
